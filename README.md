# Books Web Scraper

A Python-based web scraping project that extracts book data from Books to Scrape website and performs data analysis.

## Project Overview

This project scrapes book information including title, price, rating, and availability from [books.toscrape.com](http://books.toscrape.com/), a website specifically designed for web scraping practice. The scraped data is stored in a CSV file for further analysis.

## Features

- Scrapes 1000+ books from 50 pages
- Extracts book title, price, rating, and availability
- Handles pagination automatically
- Error handling with try-except blocks
- Exports data to CSV format
- Clean and efficient code structure

## Technologies Used

- Python 3.x
- BeautifulSoup4 (for HTML parsing)
- Requests (for HTTP requests)
- Pandas (for data manipulation)
- NumPy (for handling missing values)
- lxml (for parsing)
